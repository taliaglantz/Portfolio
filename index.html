Willy Wonka's River Run

Solo project | 8 days

This was my first project created during General Assembly’s Software Engineering Immersive Course.
The goal: as a solo project, build a grid-based browser game using HTML, CSS and Vanilla JavaScript.

Willy Wonka’s River Run is based on the 1981 game Frogger. The objective is for the player to use the arrow keys to move a character across Willy Wonka’s chocolate river. They do this by jumping on moving treats and landing on rainbow belts. At the beginning of the game, the player has the option to pick both a character and a level.

Filmojo

Pair-Programmed | 2 days

This was my second project created during General Assembly’s Software Engineering Immersive Course.
The goal: pair-programme the build of a React application that consumes a public API.

Filmojo is an app that uses two IMDb APIs to list the top 250 films. Users have the ability to view the full details of the films they select, along with the ability to add them to a wishlist and use the shuffle feature. The app was created in a 48-hour pair-coded hackathon and it was my first experience of using public APIs and React.
